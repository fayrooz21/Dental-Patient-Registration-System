# Dental Patient Registration System

## Overview
A web-based patient registration and management system developed during my internship to simplify patient registration and administrative tasks in a dental clinic.

## Features
- Patient registration
- Patient records management
- Admin dashboard
- Search by patient name or ID
- Approve or reject patient requests
- Update patient information

## Technologies
- ASP.NET Web API
- C#
- SQL Server
- HTML
- CSS
- JavaScript

## My Responsibilities
- Developed and maintained system features
- Designed and managed the SQL database
- Implemented patient registration and management functions
- Tested and improved system performance

## Project Status
Completed during my internship at the University Dental Hospital.
